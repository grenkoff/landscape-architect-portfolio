# Design System — Landscape Architect Portfolio

## Philosophy

Визуальный стиль вдохновлён сайтами
[Piet Oudolf](https://oudolf.com/) и [Dan Pearson Studio](https://www.danpearsonstudio.com/).
Строгий минимализм, монохромная палитра, акцент на фотографии.
Никакого декора — дизайн не должен конкурировать с изображениями.

### Ключевые принципы

- **Фотография — главный элемент.** Всё остальное — рамка.
- **Монохромность.** Чёрный, белый, серый. Без цветных акцентов.
- **Никакого декора.** Без divider-линий, без иконок, без gradient-оверлеев, без рамок.
- **Воздух.** Щедрые отступы, не бояться пустого пространства.
- **Плоский дизайн.** Без теней, без скруглений, без border на карточках.
- **Простая типографика.** Один шрифт, чистый sans-serif.

## Color Palette

| Token      | Hex       | Использование                     |
|-----------|-----------|-----------------------------------|
| `white`   | `#FFFFFF` | Фон основных секций               |
| `light`   | `#F5F5F3` | Фон альтернативных секций         |
| `border`  | `#E5E5E5` | Тонкие разделители (если нужны)   |
| `muted`   | `#999999` | Подписи, мета-информация          |
| `text`    | `#313131` | Основной текст                    |
| `black`   | `#000000` | Заголовки, nav links              |

### Правила

- Никаких цветных акцентов (зелёный, золотой и т.д.)
- Фон: только `white` или `light`, чередуются
- Ссылки: `black`, hover → `muted`
- Текст поверх фото: `#FFFFFF` или `#F5F3F0`

## Typography

### Шрифт

Один шрифт для всего контента: **Montserrat** (sans-serif).
Header на hero: **Courier New** (сохраняем как часть бренда).

### Размеры

| Элемент            | Size             | Weight | Letter-spacing | Transform |
|-------------------|------------------|--------|----------------|-----------|
| H2 (заголовок секции) | 0.75rem (12px) | 500    | 0.2em          | uppercase |
| H3 (название проекта) | 1rem (16px)    | 500    | 0.05em         | —         |
| Body              | 0.9375rem (15px) | 300    | 0.01em         | —         |
| Caption/meta      | 0.75rem (12px)   | 400    | 0.05em         | —         |
| Nav link          | 0.8125rem (13px) | 500    | 0.1em          | uppercase |
| Brand (header)    | 1.75rem          | 400    | 0.15em         | —         |

### Правила типографики

- Заголовки секций — маленькие, uppercase, разрежённые (как подпись, не как заголовок)
- Body текст — light weight (300), хороший line-height (1.8)
- Никаких serif-шрифтов, никакого Cormorant Garamond
- Никаких декоративных label-подписей над заголовками

## Spacing

| Контекст                   | Значение        |
|---------------------------|-----------------|
| Между секциями (py)        | `py-20` (5rem)  |
| Заголовок → контент        | `mb-12` (3rem)  |
| Между карточками (gap)     | `gap-1` или 0   |
| Контейнер padding          | `px-6 lg:px-12` |
| Max-width контейнера       | `max-w-6xl`     |

## Components

### Header (Home page)

Сохраняем как есть — два режима (transparent на hero, fixed при скролле).
Courier New branding — часть айдентики.

### Hero Section

- 100vh, object-fit cover, brush-mask на нижнем крае.
- Без текста поверх (только header).

### Section Header

Просто текст: маленький, uppercase, letter-spacing 0.2em, weight 500.
Без divider-линий, без иконок, без подзаголовков.

### About Section

- Два столбца на десктопе: фото + текст
- Фото без рамок (никаких gold-frame)
- Текст: light weight, большой line-height
- Заголовок секции — маленький uppercase текст

### Project Card (галерея на главной)

- Сетка: `grid-cols-1 md:grid-cols-2`, минимальный gap или без
- Фото на весь блок, aspect-ratio 3:2
- Hover: мягкий scale(1.02), название появляется внизу
- Overlay: полупрозрачный чёрный gradient снизу (не зелёный)
- Текст: белый, простой sans-serif

### Buttons

Минимальные: тёмный фон (#313131), белый текст, полностью скруглённые (pill).
`padding: 0.667em 1.333em; border-radius: 9999px;`

## Image Treatment

- `object-fit: cover`, никогда не искажать пропорции
- `loading="lazy"` для всех кроме hero
- Без фильтров, рамок, теней, border
- Aspect ratio проектов: 3:2

---

## Animations

| Элемент          | Свойство  | Длительность | Easing |
|-----------------|-----------|-------------|--------|
| Project img     | scale     | 0.6s        | ease   |
| Project overlay | opacity   | 0.4s        | ease   |
| Links hover     | color     | 0.2s        | ease   |

Никаких bounce, elastic, translateY. Только opacity и scale.

---

## Anti-patterns

- Без цветных акцентов (зелёный, золотой, и т.д.)
- Без декоративных divider-линий
- Без иконок и SVG-декораций
- Без serif-шрифтов в контенте
- Без теней и border на карточках
- Без gradient-фонов на секциях
- Без label-подписей над заголовками ("About", "Our Work")
- Без кнопок "View All" с рамками
